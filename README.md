# mas-bay

Aplikasi Android untuk membeli pulsa dengan UI chatbot.

## Implemented features:
- Chatting dengan bot menggunakan text.
- Hint button untuk chatting tanpa mengetik.
- Beli pulsa dengan transfer ke rekening BNI.
